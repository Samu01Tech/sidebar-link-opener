# <img src="public/icons/icon48.png" width="45" align="left"> Sidebar Link Opener

This is a Chrome extension designed to enhance your experience with the Sidebar Newsletter. The Sidebar Link Opener opens all the links from the Sidebar Newsletter in separate tabs, saving you the time and effort of manually opening each link.

## Features

- Opens all links from the Sidebar Newsletter in separate tabs.
- Lightweight and easy to use.
- Customizable settings to suit your browsing preferences.

## Usage

1. Open your Sidebar Newsletter.
2. Click the extension icon to open all the links in separate tabs.
